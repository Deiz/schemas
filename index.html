JSON Schemas for Data Protocol Formats

This project provides schemas for several of the simple data formats published as part of the Data Protocols effort.

The schemas have been implemented using the JSON Schema specification which provides a simple declarative format for describing the structure of JSON documents.

The following schemas have been created:

  • datapackage-schema.json -- datapackage.json package files
  • jsontable-schema.json -- JSON Table Schemas objects
  • csvddf-dialect-schema.json -- for validating CSV Dialect Description Format dialect objects

The schemas can be used as standalone schemas or combined to carry out validation of more complex documents, e.g. those that conform to the Simple Data Format specification.
